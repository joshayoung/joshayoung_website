# Website for joshayoung.com

### Installation Instructions
* Install brew:
  * `https://brew.sh`
* Install Node:
  * `brew install node`
* Install Gatsby (not sure if this is necessary)
  * `brew install gatsby-cli`
* Run Install
  * `npm install --global yarn`
  * `yarn install`

### Update Packages
  * `yarn update`

### Start Site:
- npm run develop
- Website Runs At: http://localhost:8080

### Deploy Updates
- `npm run deploy`
